# Rock Paper Scissors Game

This project is a simple Rock Paper Scissors game created using HTML, CSS, and JavaScript.  
The user can choose rock, paper, or scissors and play against the computer.

## Features
- User can select rock, paper, or scissors
- Computer randomly generates its choice
- Displays winner (user, computer, or draw)
- Interactive and simple game interface

## Technologies Used
- HTML
- CSS
- JavaScript
